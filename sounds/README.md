# คำแนะนำการเพิ่มไฟล์เสียง

โฟลเดอร์นี้ใช้เก็บไฟล์เสียงสำหรับแอปพลิเคชัน Math Kid's

## ไฟล์เสียงที่แนะนำ

### 1. เสียงประกอบ (Sound Effects)
- `success.mp3` - เสียงเมื่อตอบถูก (เช่น เสียงปรบมือ, เสียงสำเร็จ)
- `wrong.mp3` - เสียงเมื่อตอบผิด (เช่น เสียงแจ้งเตือนเบาๆ)
- `button_click.mp3` - เสียงเมื่อกดปุ่ม (เช่น เสียงคลิกเบาๆ)

### 2. เพลงพื้นหลัง (Background Music) ⭐ สำคัญ
- `background_music.mp3` - เพลงพื้นหลังที่เล่นอัตโนมัติเมื่อเข้าสู่หน้าหลัก
  - **แนะนำ**: เพลงสำหรับเด็ก, เพลงสนุกสนาน, เพลงบรรเลง (instrumental)
  - **ความยาว**: 1-3 นาที (จะเล่นวนลูปอัตโนมัติ)
  - **ตัวอย่าง**: เพลงเด็ก, เพลงบรรเลงเบาๆ, เพลงสนุกสนาน

## วิธีการหาไฟล์เสียง

### ตัวเลือกที่ 1: ดาวน์โหลดจากเว็บไซต์ฟรี
- [Freesound.org](https://freesound.org/) - ฟรี, ต้องสมัครสมาชิก
- [Zapsplat](https://www.zapsplat.com/) - ฟรี, ต้องสมัครสมาชิก
- [Mixkit](https://mixkit.co/free-sound-effects/) - ฟรี, ไม่ต้องสมัครสมาชิก
- [Pixabay](https://pixabay.com/sound-effects/) - ฟรี, ไม่ต้องสมัครสมาชิก

### ตัวเลือกที่ 2: สร้างเสียงเอง
- ใช้แอปบันทึกเสียงในโทรศัพท์
- ใช้โปรแกรมตัดต่อเสียง เช่น Audacity (ฟรี)

## ข้อกำหนดไฟล์เสียง

- รูปแบบไฟล์: `.mp3` (แนะนำ) หรือ `.wav`
- ความยาว: 
  - เสียงประกอบ: 0.5 - 2 วินาที
  - เพลงพื้นหลัง: ตามต้องการ
- ขนาดไฟล์: พยายามให้เล็กที่สุด (ใช้อัตราบีบอัดที่เหมาะสม)

## หมายเหตุ

- หากไม่ใส่ไฟล์เสียง แอปจะยังทำงานได้ปกติ แต่จะไม่มีเสียง
- เสียงจะเล่นก็ต่อเมื่อมีการเปิดเสียงในหน้าตั้งค่าเท่านั้น
- **เพลงพื้นหลัง**: จะเล่นอัตโนมัติเมื่อเข้าสู่หน้าหลัก (ถ้าเปิดเพลงในหน้าตั้งค่า)
- เพลงจะเล่นวนลูป (loop) อัตโนมัติ
- ระดับเสียงเพลงจะเบากว่าเสียงประกอบเล็กน้อย (50%) เพื่อไม่ให้รบกวนการใช้งาน
