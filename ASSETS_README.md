# ไฟล์ที่จำเป็นสำหรับแอป Math Kids

## โฟลเดอร์ IMAGE/
กรุณาเพิ่มไฟล์รูปภาพต่อไปนี้ในโฟลเดอร์ `IMAGE/`:

1. `LOGO.png` - โลโก้แอป
2. `learn_button.png` - ปุ่มการเรียนรู้
3. `settings_button.png` - ปุ่มการตั้งค่า
4. `addition.png` - ไอคอนการบวก
5. `subtraction.png` - ไอคอนการลบ
6. `multiplication.png` - ไอคอนการคูณ
7. `division.png` - ไอคอนการหาร

## โฟลเดอร์ sounds/
ไฟล์เสียงที่จำเป็น:
- `ting.mp3` - เสียงตอบถูก/ผิด
- `bgsound.mp3` - เพลงพื้นหลัง

## วิธีติดตั้ง
1. วางไฟล์รูปภาพในโฟลเดอร์ `IMAGE/`
2. วางไฟล์เสียงในโฟลเดอร์ `sounds/`
3. รันคำสั่ง `flutter pub get`
4. รันแอปด้วย `flutter run`
